Terminal Command Guide

Referencia cruzada interactiva para Windows y Linux

Acción Windows (CMD/PS) Linux (Bash) Explicación
Listar Archivos
dir
ls
Muestra los elementos dentro de la carpeta actual.
Crear carpeta
mkdir
mkdir
 Crea un nuevo directorio o carpeta.
Mover o Renombrar
move
mv
 Cambia la ubicación o el nombre de un archivo/carpeta.
Configuración IP
ipconfig
ip addr
 Muestra detalles de red y direcciones IP.
Prueba de conexión (Ping)
ping
ping
 Verifica si un host remoto responde.
Limpiar Pantalla
cls
clear
 Limpia todo el texto de la terminal.
Matar proceso
taskkill /f /im
kill -9
 Fuerza el cierre de una aplicación.
Apagar equipo
shutdown /s
poweroff
 Cierra el sistema operativo y apaga el hardware.
